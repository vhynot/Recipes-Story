# [Recipes-Story](https://recipes-story.netlify.app/)

## Overview

Recipes Story allows user to store particular recipes in local storage with additional options of editing and deleting them.

### Technology/tools

-   ReactJS
-   styled-components
-   Context API
-   Hooks
-   React Router
-   VSC
-   Prettier
-   ESLint
-   uuid

### Deployment

```bash
npm i
npm start
npm run build
```
