# [Recipes-Story](https://recipes-story.netlify.app)

## Overview

Recipes Story allows user to store in local storage, edit and delete particular recipes.

### Technology/tools

-   ReactJS
-   styled-components
-   Context API
-   Hooks
-   React Router
-   VSC
-   Prettier
-   ESLint
-   uuid

### Deployment

```bash
npm i
npm start
npm run build
```
